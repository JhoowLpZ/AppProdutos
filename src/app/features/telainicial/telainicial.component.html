Saudações, bem-vindo a minha aplicação desafio!

Este sistema foi desenvolvido com o objetivo de solucionar um case proposto durante o processo seletivo do Itaú!

Objetivo: Criar uma aplicação no modelo SPA usando Angular (8 ou mais recente) para consulta e cadastro de produtos.

Funcionalidades
  • Consulta de produtos: Tela onde serão apresentados todos os produtos cadastrados possibilitando a pesquisa por código e/ou categoria (opções: atacado, varejo, internacional, todos),. Deve exibir uma lista contendo código do produto, nome, categoria.
  • Inclusão: Tela para permitir a inclusão de um produto novo, permitindo informar código (único e não deve repetir), nome e categoria (deve ser uma lista pré-definida). Deve ter as opções de salvar ou voltar para a tela de consulta.
  • Alteração: Tela para permitir alterar nome e categoria de um produto já existente. Deve ter as opções de salvar ou voltar para a tela de consulta.
  • Exclusão: Excluir um determinado produto da lista apresentada na tela de consulta, solicitando a confirmação do usuário.
